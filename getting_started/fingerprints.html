Display all fingerprints for a certain serverÔÉÅ

To display all fingerprints for a certain server, you can use the following command on your local machine (Linux or macOS):

$ ssh-keyscan login.nird.sigma2.no | ssh-keygen -l -f - -E md5
$ ssh-keyscan login.nird.sigma2.no | ssh-keygen -l -f - -E sha256
