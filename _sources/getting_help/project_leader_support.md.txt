

# Project leader support

## Manage user access

As a project leader or executive officer, you can manage the user access to 
your project from <https://www.metacenter.no/>.

For more details follow the documentation at
<https://www.sigma2.no/manage-user-access>.


## Applying for extra compute allocations

It is possible to apply for extra allocations for existing projects during an
on-going allocation period. For more details follow the documentation at
<https://www.sigma2.no/extra-allocation>.


## Storage quota for projects

For the details of storage quota, follow {ref}`project-area`
about documentation on **project area** quotas.
